# 📤 SendPai — Dateien senden wie ein Senpai!

**Dein Gerät wird zum Server. Link teilen. Fertig.**

Kein Cloud. Kein Account. Kein Upload-Limit. Peer-to-Peer. Direkt.

## So geht's

1. **SendPai starten**
2. **Dateien/Ordner reinziehen**
3. **BEREITSTELLEN drücken**
4. **Link kopieren → an Empfänger senden**
5. **Empfänger öffnet Link im Browser → Download!**
6. **STOPPEN wenn fertig**

## Features

- 📁 **Dateien & Ordner** — einzeln oder als ZIP
- 📦 **Eigener Paketname** — ZIP heißt wie DU willst
- 🏠 **Lokales Netz** — sofort nutzbar im gleichen WLAN
- 🌐 **Internet** — mit Port-Forwarding weltweit erreichbar
- 🔒 **Kein Cloud** — Daten gehen DIREKT vom Sender zum Empfänger
- 📱 **Browser-Download** — Empfänger braucht KEINE App
- 🖥️ **Cross-Platform** — Linux, Windows, macOS
- ⏹️ **Ein-Klick-Schutz** — STOPPEN = sofort alles dicht

## Internet-Nutzung

Für Zugriff außerhalb des lokalen Netzwerks:

1. Router öffnen (meist `192.168.0.1`)
2. Port-Forwarding einrichten:
   - Externer Port: `7777`
   - Interner Port: `7777`
   - Interne IP: Deine PC-IP (steht in SendPai)
   - Protokoll: TCP
3. Internet-Link in SendPai kopieren → teilen!

## Sicherheit

⚠️ **Teilen Sie den Link nur mit dem gewünschten Empfänger!**

- Solange BEREITSTELLEN aktiv: Jeder mit dem Link kann zugreifen
- STOPPEN drücken = sofort alles dicht, Link funktioniert nicht mehr
- Keine Daten werden gespeichert oder weitergeleitet
- Direkte Peer-to-Peer Verbindung
